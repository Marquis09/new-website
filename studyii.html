Working Class Heroes Website Redesign - Web & UX Case Study

Improving the User Experience & Mobile Usability for a Seamless Shopping Journey

Problem

The original Working Class Heroes website had several usability and design flaws:

  • Lack of clear navigation and confusing homepage design.
  • Missing footer with important links like contact and social media.
  • Poor search functionality without error prevention or suggestive prompts.
  • Inconsistent layout, particularly with product availability and checkout process.

These issues created friction in the user's journey, making it harder to browse and shop efficiently.

Design Goals

The primary goal of the redesign was to improve the overall user experience, making the website more intuitive and efficient for users:

  • Improve homepage navigation by adding clear sections and a welcoming introduction.
  • Make the website mobile-responsive for on-the-go users.
  • Incorporate a clear and functional search bar with error prevention and suggestion prompts.
  • Revamp product pages by adding real-time stock availability and improving checkout processes.

Design Solutions

Solution 1: Improved Homepage Design

To address the problem of a cluttered homepage, I introduced a carousel for the images, along with a welcoming introduction to guide users.




The carousel allowed for a cleaner design, offering the user a chance to view various categories without overwhelming them with too much imagery at once.

Solution 2: Enhanced Search Functionality

The search functionality was enhanced to include error prevention and suggestive prompts, ensuring users could easily find what they were looking for without frustration.




Solution 3: Product Page Improvements

I revamped the product pages by adding stock availability before selecting the size, providing users with clearer information.




Solution 4: Streamlined Checkout Process

I simplified the checkout process by removing redundant pop-up windows and consolidating all checkout information into one page.
