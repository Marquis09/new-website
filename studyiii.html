MMU TV Website - Web Development & UX Case Study

Building a "CREATIVE PLATFORM" for the MMU Community to showcase and explore media projects.

Project Details

  • My Role: Web Developer, UX Designer
  • Duration: 3 Weeks
  • Tools: HTML5, CSS3, Bootstrap, Figma
  • Project Type: University Project / "COMMUNITY PLATFORM"

1. Project Overview & Introduction

The MMU TV website was created as a platform to showcase "CREATIVE PROJECTS" from Manchester Metropolitan University (MMU) students and staff. The core goal was to make it easy for users to "EXPLORE VIDEOS", sign up, and engage with the MMU community.

The Challenge: To design and build a fully functional, "MOBILE-FRIENDLY" website from scratch that could host diverse media content while maintaining an intuitive user experience for both student creators and casual viewers.

2. Challenges & Design Goals

Key Challenges

  • Layout: Lack of a clear, "MODERN LAYOUT" for displaying diverse video content.
  • Accessibility: Ensuring the website was mobile-friendly and met "ACCESSIBILITY STANDARDS".
  • Search: Creating a simple but "FUNCTIONAL SEARCH FEATURE" for filtering video content.
  • User Needs: Developing an intuitive user experience that caters to "BOTH STUDENTS AND STAFF".

Primary Design Objectives

  • Responsiveness: Build a responsive, "MOBILE-FIRST DESIGN" for easy access on all devices.
  • Navigation: Develop an "INTUITIVE NAVIGATION SYSTEM" to quickly find and view videos.
  • Visual Appeal: Create visually appealing and "ENGAGING CONTENT SECTIONS" to showcase creative work.
  • Simplicity: Design the sign-up and interaction processes for "MAXIMUM SIMPLICITY".

3. Development Process & Key Solutions

The website was built using HTML5, CSS3, and the Bootstrap framework, focusing on the "FRONT-END DEVELOPMENT" and visual presentation.

The homepage features a full-width hero image, a brief introduction, and a section highlighting "FEATURED VIDEOS". The layout is simple, with easy navigation to key pages like "CATEGORIES" and "SIGN UP".

4. Final Thoughts & Next Steps

This project was an exciting opportunity to apply my "FRONT-END DEVELOPMENT SKILLS" and create a user-centric website that serves the MMU community. With responsive design, accessibility, and simple navigation, the MMU TV website provides a "SEAMLESS AND ENGAGING EXPERIENCE" for users across all devices.

Future Steps

  • User Accounts: Implement full user authentication and "PERSONALIZED DASHBOARDS".
