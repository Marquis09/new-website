MMU TV Website - Web Development & UX Case Study

Building a Creative Platform for the MMU Community

Introduction

The MMU TV website was created as a platform to showcase creative projects from Manchester Metropolitan University (MMU) students and staff. The goal of the website was to make it easy for users to explore videos, sign up, and engage with the MMU community.

Challenges

When starting this project, there were several challenges to overcome, including:

  • Lack of a clear, modern layout for displaying content.
  • Ensuring the website was mobile-friendly and accessible.
  • Creating a simple but functional search feature for video content.
  • Developing an intuitive user experience that caters to both students and staff.

Design Goals

The primary objectives for the website redesign were:

  • Build a responsive, mobile-first design for easy access on all devices.
  • Develop an intuitive navigation system that allows users to quickly find and view videos.
  • Ensure the website meets accessibility standards for users with disabilities.
  • Create visually appealing and engaging content sections to showcase MMU‚Äôs creative work.

Development Process

The website was built using HTML5, CSS3, and the Bootstrap framework for responsive design. Below are the key steps taken during development:

Homepage Design

The homepage features a full-width hero image, a brief introduction, and a section highlighting featured videos. The layout is simple, with easy navigation to key pages like "Categories" and "Sign Up".




Mobile Responsiveness

One of the main focuses was to make the site fully responsive. By utilizing Bootstrap‚Äôs grid system, the website automatically adjusts for devices of any size, ensuring a seamless experience on phones, tablets, and desktops.




Search Functionality

To enhance user experience, I implemented a simple search bar that suggests results as users type. This feature makes it easier to find specific videos and categories.




Sign-Up Page

The Sign-Up page allows users to register easily with a form designed for simplicity. The page also includes social media links to encourage users to connect and stay updated on MMU TV's latest content.
