Project Introduction

This project involved creating a responsive personal portfolio website to showcase skills and past work. It was built using HTML, CSS, and JavaScript.

Design Process

The design process began with creating wireframes in Figma, followed by designing the website layout and user interface, ensuring a mobile-first approach. Feedback from peers was gathered throughout the process.

Resources

Tools used include Figma for wireframing, Adobe Photoshop for asset creation, and Sublime Text for coding. Extensive online research on best practices for portfolio sites was also conducted.

Code Implementation

The website was coded using HTML for structure, CSS for styling, and JavaScript for interactive elements such as form validation and smooth scrolling. It is fully responsive, adapting seamlessly across devices.

Conclusion

The final result was a visually appealing and highly functional portfolio that demonstrates my front-end development skills and showcases my work effectively.
