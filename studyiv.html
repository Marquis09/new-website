Project H: Community Hub Redesign

From Text-Heavy Archives to Bento Grid Architecture

Executive Summary

  • Challenge: Modernize a text-dense legacy site without changing the familiar layout structure.
  • Solution: Lumia-inspired Bento Grid UI that breaks information into modular, scannable tiles.
  • Milestone: Proposed design preferred over incumbent professional agency work by organizational leadership.

Project Details

  • My Role: UX Designer UI Innovator
  • Tools & Tech: HTML5 CSS3 UX Strategy
  • Project Type: Community Hub Redesign
  • Constraints: Legacy Layout Retention | Sanitized Client Content
Process Index: 1. The Vision | 2. Bento UI | 3. Evolution | 4. Prototype | 5. Next Steps | 6. Reflection

1. The Vision: Same Layout, Better UI

I set out to prove that Information Architecture doesn't have to be overwhelming. I kept the structural layout the client was used to, but completely overhauled the Visual Interface. By removing the "Wall of Text," I created an environment that is action-oriented and accessible to all ages.

"During the final presentation, the organization’s leadership highlighted how the reduction in text-heavy sections made the core message more impactful and the site significantly faster to navigate."

2. UI Innovation: The "Lumia" Bento Transformation

The defining change was the implementation of a Bento Box UI. This modular approach allowed me to keep the same site structure while drastically improving the UX:

  • ◈ Modular UI: Information is broken into scannable, bite-sized tiles.
  • ◎ Touch Targets: Expanded touch targets designed for seamless navigation on mobile and for elderly users.
  • ✦ Zero Bloat: 60% reduction in on-screen text clutter for improved cognitive clarity.
📊 Key Design Impact: Achieved a 60% reduction in on-screen text density while preserving 100% of core informational hierarchy.

5. Next Steps & Future Iterations

  • Accessibility Audit (WCAG 2.1 AA): Perform high-contrast theme testing to ensure full readability for low-vision users.
  • Content Management Integration: Build custom Bento Grid templates within a headless CMS to allow non-technical stakeholders to add news tiles seamlessly.
  • User Analytics Integration: Track click-through rates on individual Bento tiles to continuously optimize grid item placement.

6. Technical & Ethical Reflection

To respect the privacy of the organization, identifying content has been sanitized. This project highlights my ability to take an agency-managed "wall of text" and transform it into a modern Modular UI that satisfies both user needs and stakeholder goals.
