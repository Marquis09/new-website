Project H: Community Hub Redesign

From Text-Heavy Archives to Bento Grid Architecture

Executive Summary

  • Challenge: Modernize a text-dense legacy site without changing the familiar layout.
  • Solution: Lumia-inspired Bento Grid UI.
  • Milestone: Design preferred over incumbent professional agency work.

Project Details

  • My Role: UX Designer, UI Innovator
  • Tools: HTML, CSS, UX Strategy
  • Project Type: Community Hub Redesign

1. The Vision: Same Layout, Better UI

I set out to prove that Information Architecture doesn't have to be overwhelming. I kept the structural layout the client was used to, but completely overhauled the Visual Interface. By removing the "Wall of Text," I created an environment that is action-oriented and accessible to all ages.

"During the final presentation, the organization’s leadership highlighted how the reduction in text-heavy sections made the core message more impactful and the site significantly faster to navigate."

2. UI Innovation: The "Lumia" Bento Transformation

The defining change was the implementation of a Bento Box UI. This modular approach allowed me to keep the same site structure while drastically improving the UX:

  • ◈ Modular UI: Information is broken into scannable tiles.
  • ◎ Touch Targets: Perfect for mobile and elderly users.
  • ✦ Zero Bloat: 60% less text on screen for clarity.
