## 🛠️ The Developer Pipeline (Tech Stack)
Because I bridge the gap between UI/UX design and computer science fundamentals, this entire portfolio was hand-coded without generic builders:

- **Structure:** Semantic HTML5 ensuring clean document outline structures.
- **Styling:** Responsive, custom CSS3 leveraging Flexbox/Grid layouts and smooth hover states.
- **Interactions:** Vanilla JavaScript handling fluid interface transitions.
- **Accessibility Framework:** Contrast boundaries and layout scalability built around WCAG compliance parameters.

---

## 📈 Key UI/UX Systems Implemented
- **Device Versatility:** Custom fluid grids that dynamically scale from a 320px mobile viewport up to widescreen displays.
- **Interactive States:** Explicit visual responses (active, focus, hover) across all buttons and menus to elevate usability.
